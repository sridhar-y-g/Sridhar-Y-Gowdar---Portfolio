# Sridhar Y Gowdar - Portfolio Website

## Features

- Responsive design that works on all devices
- Interactive UI with smooth animations and transitions
- Multiple pages with detailed project information
- Skills categorization with progress indicators
- Timeline-based education history
- Project showcase with detailed individual project pages
- Contact form with validation
- Custom cursor follower effect
- Animated section transitions

## Technologies Used

- HTML5
- CSS3 (with animations and keyframes)
- JavaScript (for interactivity and animations)
- Font Awesome (for icons)
- Google Fonts

## Pages

- **Home**: Main landing page with hero section
- **About**: Personal information and bio
- **Skills**: Technical skills categorized by type
- **Education**: Academic history in timeline format
- **Projects**: Portfolio of completed projects
- **Project Details**: Individual pages for each project
- **Certifications**: Professional certifications
- **Contact**: Contact form and information
